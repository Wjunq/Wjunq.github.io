ubuntu系统

ubuntu操作系统是linux的一种

linux系统中：

ubuntn改中文

Ubuntu系统下面可以配置中文

1，设置->系统设置

2，选择区域与语言

单击管理已安装的语言
3，单击添加或者删除语言

4，选择简体中文

5，单击应用等待安装成功

6，在菜单和窗口的语言框中，将中文拖拽放入第一个位置

7，重启系统中文才可以使用

8，使用默认英文目录（需要注意，否则使用起来非常麻烦）

9，此时系统已经可以显示中文了，但是还不能输入中文

添加输入源

命令

命令 解释
ls 查看当前目录下的文件信息
tree 文件名 以树状的方式显示目录信息
pwd 查看当前目录路径
clear 清屏
ps -e|grep apt 查看一下apt-get的相关进程
touch 文件名 创建指定文件
mkdir 目录名 创建目录
rm 文件名或者目录名 删除文件或目录
rmdir 目录名 删除空目录
cp 复制
cat 查看文件的内容
mv 移动文件
ll 以列表形式显示所有的文件（包括隐藏文件）

切换目录目录

命令 解释
cd目录 切换指定的目录
cd ~ 切换到当前用户的主目录
cd .. 切换上一级目录
cd . 切换到当前目录
cd - 切换到上一次的目录

终端命令格式的组成

command -[-options] [parameter]

  • –help命令 ls –help
  • man命令 man ls

ls命令

命令选项 说明
-l 以列表的形式显示文件
-h 以大小单位显示，默认字节
-a 显示隐藏文件

mkdri

命令选项 说明
-p 创建所依赖的文件夹

rm

命令选项 说明
-r 递归删除目录及其内容
-f 强制删除，忽略不存在的文件
-i 交互式提示
-d 删除空目录
